# Лаба 03

## 3. Описание реализованной иерархии классов

### Базовый класс
`Product` — общий класс для всех сущностей. Содержит общие поля, валидацию, расчет цены, покупку, активацию и деактивацию.

### Дочерние классы
`LiquidProduct` — жидкий товар. Дополнительные атрибуты: `volume_liters`, `bottle_fee`. Переопределяет `calculate_price()` и `__str__()`.

`TangibleProduct` — физический предмет. Дополнительные атрибуты: `weight_kg`, `logistics_fee`. Переопределяет `calculate_price()` и `__str__()`.

`ServiceProduct` — услуга или работа. Дополнительные атрибуты: `hours`, `complexity`. Переопределяет `calculate_price()` и `__str__()`.
