# Лаба 2

## Описание класса контейнера ProductCatalog:

## Назначение

Класс `ProductCatalog` представляет собой контейнер для хранения и управления объектами класса `Product`.  
Обеспечивает безопасную работу с коллекцией: добавление, удаление, поиск, сортировку и фильтрацию объектов.

---

## Конструктор

### `__init__(items=None)`

Создает новую коллекцию.

**Параметры:**
- `items` — итерируемый набор объектов `Product` (необязательно)

**Поведение:**
- инициализирует пустую коллекцию
- при передаче `items` добавляет элементы через `add()`

---

## Основные методы

### `add(item)`

Добавляет объект в коллекцию.

**Параметры:**
- `item` — объект `Product`

**Исключения:**
- `TypeError` — если передан не `Product`
- `ValueError` — если объект с таким именем уже существует

---

### `remove(item)`

Удаляет объект из коллекции.

**Параметры:**
- `item` — объект `Product`

**Исключения:**
- `TypeError` — если передан не `Product`
- `ValueError` — если объект не найден

---

### `remove_at(index)`

Удаляет объект по индексу.

**Параметры:**
- `index` — целое число

**Возвращает:**
- удалённый объект `Product`

**Исключения:**
- `TypeError` — если индекс не число
- `IndexError` — если индекс вне диапазона

---

### `get_all()`

Возвращает копию списка всех объектов.

**Возвращает:**
- `list[Product]`

---

### `find_by_name(name)`

Ищет объект по имени.

**Параметры:**
- `name` — строка

**Возвращает:**
- объект `Product`, если найден
- `None`, если не найден

**Исключения:**
- `TypeError` — если имя не строка

---

## Методы сортировки

### `sort(key=None, reverse=False)`

Сортирует коллекцию.

**Параметры:**
- `key` — функция сортировки
- `reverse` — обратный порядок

**Возвращает:**
- текущий объект `ProductCatalog`

---

### `sort_by_name(reverse=False)`

Сортирует по имени.

---

### `sort_by_price(reverse=False)`

Сортирует по цене с учетом скидки.

---

## Методы фильтрации

### `get_active()`

Возвращает новую коллекцию активных товаров.

---

### `get_available()`

Возвращает новую коллекцию доступных товаров  
(активных и с количеством > 0).

---

### `get_expensive(min_price)`

Возвращает новую коллекцию товаров дороже заданной цены.

**Параметры:**
- `min_price` — минимальная цена

---

## Специальные методы

### `__len__()`

Возвращает количество элементов в коллекции.
