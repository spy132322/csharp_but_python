### 1. Интерфейс `Printable`

**Контракт:** объект может быть представлен в виде строки.

```python
from abc import ABC, abstractmethod

class Printable(ABC):
    @abstractmethod
    def to_string(self) -> str:
        """Возвращает строковое представление объекта."""
        pass
```

### 2. Интерфейс Comparable

Контракт: объект может сравниваться с другим объектом.

```python
class Comparable(ABC):
    @abstractmethod
    def compare_to(self, other: object) -> int:
        """
        Сравнивает текущий объект с other.
        Возвращает:
        < 0 – если текущий объект меньше,
        0   – если объекты равны,
        > 0 – если текущий объект больше.
        """
        pass
```

Реализация интерфейсов в классах

Интерфейсы Printable и Comparable реализованы в базовом классе Product (из лабораторной работы №3), а также во всех его наследниках:

- LiquidProduct – жидкий продукт (учитывает объём и тип тары);

- TangibleProduct – материальный товар (учитывает вес и логистику);

- ServiceProduct – услуга (учитывает часы работы и сложность).
